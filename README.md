# TiempoSur – App de Clima para las Ciudades de América del Sur.

Este proyecto demuestra mi capacidad para integrar APIs, manejar estados en React y aplicar diseño responsive para experiencia móvil.


# Tecnologías utilizadas

Frontend: JavaScript, HTML5, CSS3

API clima: uso de una API pública para obtener datos meteorológicos (temperatura, humedad, etc.)

Diseño: sección de selección de país/ciudad, interfaz clara y funcional, diseño responsive.

Puedes ver el proyecto funcionando aquí: [https://nahuelmndz.github.io/TiempoSur/](https://tiemposur.netlify.app/)
